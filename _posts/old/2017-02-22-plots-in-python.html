Introduction to plotting in Python¶

Today we will begin to learn how to build plots in python using the matplotlib library. This lab does not cover all of the types of plots that can be built with matplotlib; rather, it focuses on the thought process behind building a meaningful and informative graphic. For detailed instructions on matplotlib, please visit the libary's homepage.

Review of Cocoli census data¶

In lecture, we began to work with forest dynamics data collected from a tropical dry forest at Cocoli, near the Panama Canal. Each row in the file cocoli.txt information about a single individual in this forest. For a thorough guided exploration of the data, please refer to Lecture 13.

We will begin by reading in the dataset and creating three dictionaries. The key for each dictionary will be the unique Tag ID used to track each tree; the values will be the size of the individual in the 1994, 1997, and 1998 census. Note: we could also have made a single dictionary with a Tag ID as key and a list of sizes as the value.

Our first plot¶

We now have three dictionaries (namely, sizes_94, sizes_97, and sizes_98) that contain tree sizes.

There are a number of exploratory visualizations we can make here, but I chose to begin with a simple histogram of sizes - in other words, what was the distribution of tree sizes? What was the maximum?

We begin by importing in the matplotlib library and setting up our data as a list:

Logging the Y-axis¶

One of the limitations of our histograms is that our Y-axis is so dominated by the smallest size class, that we can't really tell how many individuals are in any of the bigger size classes. We can get around this by logging the Y-axis.

Our figure also needs a title, and some labels- for example, what are the units on the X-axis, anyway? (Turns out that in our data, DBH is reported in mm)

In [4]:
# Add a log-axis to our histogram
plt.hist(sizes_in_94, log = True)
# an X-axis label
plt.xlabel('DBH (mm)')
# a y-axis label
plt.ylabel('Frequency')
# A main title for the graph
plt.title("Size distribution of trees in Cocoli (1994) (all stems)")
# Let's look!
plt.show()

We can finally interpret our histogram now- it looks like there is one individul each in the largest two size classes, about 10 in the third-biggest size class, and so on.

Fixing our plots¶

Our plots above look very pretty, but the dark truth of those plots is that they are factually incorrect. For example, our smallest size class (left-most bar in the histogram) is apparently at >8000 stems, but there are in fact \<8000 stems in that smallest size class.

To understand why, we need to rely on our understanding of the dataset: every time there is a -2 in a DBH column, that indicates a "hidden" tree that wasn't alive in one census, but came up before a second. Hint- see Lecture 13 for some more insight.

So, we need to get rid of all instances of -2 in our sizes_in_94 list. To achieve, this we use a python approach called list comprehension:

In [37]:
# use list comprehension to remove 
# all instances of -2 from our list
sizes_in_94[:] = (value for value in sizes_in_94 if value != -2)
